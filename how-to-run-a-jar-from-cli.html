How to Run a JAR File from Command Line

If double-clicking the JAR file doesn't work, you can run it from the command line using these steps:

Step 1: Open Command Prompt in the Downloads Folder

Navigate to your Downloads folder in File Explorer. Then, click the address bar (or use Ctrl + L) and type cmd, then press Enter. This will open Command Prompt in the current directory.

Step 2: Run the JAR File

In the Command Prompt window, type the following command:

java -jar neoforge-21.1.115-installer.jar

Then press Enter. This will execute the JAR file and launch the NeoForge installer.

Troubleshooting

  • If you see an error about Java not recognized, ensure you have Java installed. You can download it from the official Java website.
  • If the JAR file fails to launch, double-check the file name and location, and ensure you're typing the command exactly as shown.

Once the installer runs successfully, return to the main tutorial to continue with the installation process.
